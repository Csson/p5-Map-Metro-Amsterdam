# NAME

Map::Metro::Plugin::Map::Amsterdam - Map::Metro map for Amsterdam

# SYNOPSIS

    use Map::Metro;
    my $graph = Map::Metro->new('Amsterdam')->parse;

Or:

    $ map-metro.pl route Amsterdam Isolatorweg Gein
